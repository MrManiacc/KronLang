| Token Description     | Token Example |
|----------|------------|
| Single comment                 |  ``/~``          |

| Multi comment *End*            |  ``~/``          |
| String |``"hello, world"`` |
| Number | ``2323, 12.32, -3221`` |
| Identifier| ``test.testing<5.5332>`` |
